Description

WebInsight is a Chrome side-panel extension that captures page text, selections, screenshots, and PDFs while you browse. Content is tagged, stored locally in IndexedDB, and can be exported as a structured JSON package. That JSON feeds downstream tools to generate deep research papers, agent definitions, persona configurations, and knowledge graphs.

Current Functionality

Capture Modes: full page text, selection, screenshots, full-page PDF.

Tagging & Filtering: add/remove tags, filter view by tag.

Storage: IndexedDB with contentItems, tags, and contentTags.

Exports:

PDF report (human-readable).

Project JSON (for AI processing).

AI Features:

Local AI tag suggester (keyword-based).

Cloud AI hooks (Gemini API).

Integrations: optional Google Drive backup/restore.
