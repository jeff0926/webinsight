WebInsight Settings

API Configuration

Your API key enables image analysis features. It is stored locally in your browser's storage and is required for visual analysis. Get an API Key from Google AI Studio.
Security Note: While stored locally, browser storage is not perfectly secure. For production use, a backend solution is recommended.

Google Drive Integration

Required for Google Drive backup and sync features. Get a Client ID from Google Cloud Console.
Setup: Create an OAuth 2.0 Client ID and enable the Google Drive API.
